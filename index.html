Respostas

A. O atributo placeholder é uma pequena frase ou palavra, que tem o intuito de ajudar usuários a entender como devem preencher determinado formulário, inserindo um texto contextual no campo a ser respondido

B. Quando o valor do atributo type é radio ou checkbox , a presença deste atributo booleano indica que o controle é selecionado por padrão; caso contrário, este atributo é ignorado

C. No atributo value="Enviar dados", especifica-se o texto que será lido pelo leitor. e="Enviar dados", especifica-se o texto que será lido pelo leitor.

D. form> INPUT TYPE="RADIO" NAME="OPCAO" VALUE="op1"> opção1 INPUT TYPE="RADIO" NAME="OPCAO" VALUE="op2"> opção2 /form>

E. A tag label é útil no trabalho com formulários. Ela especifica qual o "rótulo" do input e ajuda na experiência do usuário durante a utilização e preenchimento do formulário, outra coisa que ajuda no preenchimento do formulario durante a utilização e o comando for, que indica dentro label por exemplo

F. O elemento HTML select (select) representa um controle que apresenta um menu de opções. As opções dentro do menu são representadas pelo elemento

G. Required = se true exibe a mensagem de erro caso o usuário deixe em branco, se false não exibe nada.

H. Definir para aonde os dados serão enviados

I. O atribuito “method” irá definir a forma que os dados serão enviados

J. form action="URL_do_script" method="post"

K. form action="URL_do_script" method="get"
